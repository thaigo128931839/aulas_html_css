ID e class

ID

o id funciona como um cpf Assim so posso ter um numero de cpf cada elemneto só pode ter um valor de id . O cpf é unico , ou seja ninguém pode ter acesso ao cpf igual ao meu . Isso siginifica que o ID tabém é unico , então não e possivel repetir o mesmo valor duas vezes .

class

o CLASS funciona de maneira oposta ao id .Diferente do id emque serve você só pode ter um valor , no class você pode ter varios valores tabem diferente do id em que não serve para uma pessoa
